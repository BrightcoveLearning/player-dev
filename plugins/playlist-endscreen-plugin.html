Overview

The Playlist Endscreen plugin allows you to display a playlist when a video ends. This playlist can be related videos, using a smart playlist, or it can be videos unrelated to the main video.

The playlist endscreen will NOT appear if

  • No playlist ID is provided
  • The playlist ID is unset at runtime
  • The plugin is disabled

The playlist endscreen will appear if

  • A playlist ID is available AND
  • The plugin is enabled

Implement using Players module

Playlist endscreens can be implemented using Video Cloud Studio's Players module. For details, see the Adding Plugins using the Players Module document.

Implement using code

To implement a plugin the player needs to know the location of the plugin code, a stylesheet if needed, the plugin name and plugin configuration options. The location of the plugin code and stylesheet are as follows:

  1. Include the plugin code:

    https://players.brightcove.net/videojs-playlist-endscreen/1.0.1/videojs-playlist-endscreen.js

  2. Include the plugin styles:

    https://players.brightcove.net/videojs-playlist-endscreen/1.0.1/videojs-playlist-endscreen.css
  3. Use the plugin name to load it into the player: playlistEndscreen

  4. An example set of options is:

    {
    "playlistId": 4602525726001,
    "title": "Watch More"
}

  5. Here is an example using the Advanced Embed implementation to associate the Playlist Endscreen plugin with a single instance of a player.

    <link href="https://players.brightcove.net/videojs-playlist-endscreen/1.0.1/videojs-playlist-endscreen.css" rel="stylesheet">
  
  <div style="max-width: 960px;">
  <video-js id="myPlayerID"
    data-account="1752604059001"
    data-player="2efXCCxGU"
    data-embed="default"
    controls=""
    data-video-id="4029697544001"
    data-playlist-id=""
    data-application-id=""
    class="vjs-fluid">
  </video-js>
</div>
<script src="https://players.brightcove.net/1752604059001/2efXCCxGU_default/index.min.js"></script>

<!-- script for the Playlist Endscreen plugin -->
<script src="https://players.brightcove.net/videojs-playlist-endscreen/1.0.1/videojs-playlist-endscreen.js"></script>

<!-- custom script -->
<script>
  videojs.getPlayer('myPlayerID').ready(function() {
    var myPlayer = this;

    // initialize the Playlist Endscreen plugin
    myPlayer.playlistEndscreen({"playlistId": 4845949311001}); 
  });
</script>

Options

You may pass in an options object to the plugin upon initialization. This object may contain any of the following options:

playlistId

playlistId
  • Type: string
  • The string representing the playlist ID to be displayed in the endscreen.

title

title
  • Type: string
  • The string to customize the endscreen title, this will also modify the button tooltip.

Setting playlist ID

There are additional ways to configure playlist ID to retrieve the playlist you want. This playlist ID can come from one of the following sources:

  • Attribute on in-page embeds: data-endscreen-playlist-id="123"
  • Query string for iframes: &endscreenPlaylistId=123
