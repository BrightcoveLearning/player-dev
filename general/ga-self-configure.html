Introduction

The configuration process consists of the following tasks:

  • Select the events to track from those supported by Brightcove Player.
  • Select the custom dimensions from those supported by Google Analytics.
  • In Google Analytics, create the selected custom dimension.
  • In Video Cloud Studio, add configuration information about your selections using the player's JSON editor.
